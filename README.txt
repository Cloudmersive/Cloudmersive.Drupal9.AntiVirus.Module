INTRODUCTION
------------
 
Cloudmersive Anti-virus Scanner integrates with the anti-virus scanner
Cloudmersive Antivirus API. Uploaded files are forwarded to the Cloudmersive
Antivirus service and checked. Infected files are blocked in the validation
routine, so they cannot be saved.

REQUIREMENTS
------------

This module requires a Cloudmersive Antivirus service.

 * Sign up for a free API key at https://cloudmersive.com
 * Free keys include 50,000 API calls/month.

INSTALLATION
------------
- Get a Cloudmersive API key at https://cloudmersive.com
* Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
   for further information.
